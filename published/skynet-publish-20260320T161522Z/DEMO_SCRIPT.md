# Demo Script (3 Minutes)

## 1) Problem Setup (20s)
"Crypto users with income and trades across jurisdictions struggle to stay compliant. Skynet automates tax-event interpretation with transparent assumptions."

## 2) Input (40s)
- Open app
- Upload sample CSV
- Choose jurisdiction (e.g., US) and tax year (e.g., 2025)

## 3) Processing (45s)
- Show normalized transactions
- Show event classification tags (income/swap/transfer/staking)
- Trigger report generation

## 4) Explainability (40s)
- Open one line item
- Show applied rule id and version
- Show formula inputs and resulting taxable value

## 5) Fallback Example (25s)
- Switch to jurisdiction with missing crypto-specific rule
- Show fallback policy activated and confidence adjusted

## 6) Output + Close (10s)
- Export report (JSON/Markdown/PDF)
- "Skynet helps users understand tax obligations faster with transparent, jurisdiction-aware AI logic."
